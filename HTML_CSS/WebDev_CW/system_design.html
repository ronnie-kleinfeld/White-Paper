Visual look of the reset-utility.css file


This document serves as a component of the tasks 3.1.2 and 3.1.12 within the coursework. Its purpose is to provide a visual representation of the external file "reset-utility.css". This CSS file is designed to ensure a consistent look and feel across the website, offering default styling for various page components, along with a defined color palette and typography system. Additionally, the external file offers CSS reset properties, facilitating the seamless integration of additional CSS solutions.



For implementation, the CSS file should be included in the head section of the HTML page for each student, using a link tag such as: <link rel="stylesheet" href="reset-utility.css">. It's essential to position this link before the student's own CSS file. By adhering to this structure, any personal CSS styling applied by the student will overwrite the default styles established by the external CSS file. This approach grants students greater freedom to imbue their pages with their unique stylistic preferences.


1) Colours

These colors are intended for consistent use across the website, ensuring a cohesive color palette throughout.


To apply these colors, students can utilize the class "bg-color-v(n)" to set the background color of an element or "text-color-v(n)" to specify the color of the text within an element.
