Styling & CSS

CSS Syntax

Keywords:

  • Rule: p {text-indent: 2em;} 2 parts Selector and Declaration
    • Selector: the HTML element
    • Declaration: the style - Property: Value;
      • Property: the property name
      • Value: the value of the style 
h2 {
    color: #800080;
    font-style: italic; 
}

Selectors

  1. By name:
    2. h1 { color: blue; }
  2. 2 rules
    3. h1 { 
          background: white; 
          color: blue; 
      }
  3. Multiple selectors
    4. h1, h2 { color: blue; } 
      
Same as:   
h1 { color: blue; }
h2 { color:blue; }
  4. Nested tags
    5. p em { color: green; }
    <body> 
    <h1>not <em>not</em> </h1>
    <p>not <em>yes</em></p>
</body>
  5. class style
    6. .official { color: blue; }
    <body> 
    <h1 class="official">Some heading </h1>
    <p class="official">Some text </p>
    <p>Some more text </p>
</body>
  6. id style
    7. #intro { color: green; }
    Example use in html follows. Note ids are used to identify one unique part of a page. 
    <body> 
    <h1 id="intro">Some heading </h1>
    <p id="intro">Some text </p>
    <p>Some more text </p>
</body>
  7. class combining
    8. div.summary { color: green; }
    <div class="summary">should be green</div>
  8. id combining
    9. div#summary { color: green; }
    <div id="summary">should be green</div>
  9. catch all
    10. * { color: blue; }
  10. link statue
    11. a: hover { background-color: silver; }

    Peoperties samples

    text-indent indents the first line. length or % and negative
    font-family font set. family-name (Times, Arial, Garamond)
    generic-name (serif,sans-serif, monospace, cursive, fantasy)
    more than one font-family to let the browser until it finds a font
    font-style oblique or italic
    font-weight Absolute values are 100, 200, 300, ... 900, 400 being same as normal
    Relative values are bold, bolder,lighter
    text-align text alignment (left, right, center, justify)
    font-size sets the size of a font. Values can be absolute, relative, length and percentage
    text-decoration adds underlining, overlining, strike-out or blink to text
    none, overline, line-through, underline and blink
    text-transform changes the case capitalize, uppercase, lowercase.

    <span> and <div> elements

    • <span> element is a grouping element. Its primary purpose is to apply CSS rules or id attributes to a block of text. It is an in-line level element, which means it is displayed in line with other text and with no line breaks.
    • <div> is a similar element but displayed on its own line, with margins above and below. It is a block-level element
