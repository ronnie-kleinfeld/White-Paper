Table Styling

<table>, <th>, <td> border, e.g.
td {border: 1px solid black;}		 border width <table border="1">
<table>, <tr>, <th>, <td> padding, e.g.
td {padding: 25px;} 		gap around contents
background-color, e.g.
table {background-color: aqua;} 		background color
width, e.g.
table {width: 80%;}		 80% of window width
width, e.g.
table {width: 500px;}		 500 pixels wide
<th>,<td> text-align, e.g.
td {text-align: right;}		 text alignment left, right, center or justify
vertical-align, e.g.
td {vertical-align: top;}		 vertical alignment top, bottom or middle
