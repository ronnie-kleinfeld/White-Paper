Headings

A heading in the text is created with the <h1> tag. There are in fact six heading tags, going from <h1> the largest to <h6> the smallest.

Formatted text HTML markup required

A H1 heading

 <h1>A H1 heading</h1>

A H2 heading

 <h2>A H2 heading</h2>

A H3 heading

 <h3>A H3 heading</h3>

A H4 heading

 <h4>A H4 heading</h4>
A H5 heading
 <h5>A H5 heading</h5>
A H6 heading
 <h6>A H6 heading</h6>
